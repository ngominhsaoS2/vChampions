*** User (bao gồm cả cầu thủ/chủ sân)
- id
- name
- email
- phone
- password
- role (vai trò: cầu thủ/manager/chủ sân)
- images (các ảnh, và ảnh avatar)
- birthday
- description (tự mô tả =))) )
- clubs (một cầu thủ có thể tham gia nhiều clubs)
- stats (một số thống kê về cầu thủ: số trận, số bàn, trung bình...)


*** Club
- id
- code
- name
- logo
- manager
- captain
- players
- stats (một số thống kê về đội bóng: số trận, số bàn, trung bình...)
- rank (xếp hạng của đội bóng)


*** Stadium
- id
- name
- images
- address
- yards (các sân nhỏ trong stadium)
    + name
    + prices
        * from
        * to
        * price
    + isReserved
    + homeClub
    + visitingClub
    + realtimeUpdate


*** Financial
- id
- stadiumId
- year
- monthReports
- yearReports
- matches
    + yard
        * name
        * startTime
        * endTime
        * price
    + homeClub
    + visitingClub
    + results
        * homeClub []
            ~ player
            ~ time score
        * visitingClub []
            ~ player
            ~ time score
